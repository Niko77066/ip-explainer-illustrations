# 默认视觉 DNA

只在选中家族或角色没有可确认动画画风时使用。存在动画画风时，以 `family-style-catalog.md` 和 `style-master.png` 为准。

- 16:9 横版正文配图。
- 纯白背景；不要纸纹、米色、渐变、阴影或复杂环境。
- 黑色细手绘线为主，线条轻微抖动，保留草图感。
- 主体占画面约 40%-60%，至少留出 35% 安静空白。
- 每张只讲一个判断、机制、状态或隐喻。
- 中文手写标注尽量控制在 3-5 个，每个 2-8 字。
- 橙色只表示主要路径或动作；红色只标问题、阈值或结果；蓝色只作补充说明。
- 风格要冷静、清爽、有判断，不做商业海报、PPT 信息图、儿童插画或复杂架构图。
- 不在左上角写“流程图、架构图、常见坑”等类型标题。

用户可以覆盖画风，但身份参考、双锚点和一致性规则仍然生效。用户认可第一张正文图后，把它记录为固定画风锚点；后续不得逐张改画风描述。
