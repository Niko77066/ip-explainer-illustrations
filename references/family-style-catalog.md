# 家族动画画风目录

目录记录画风选择逻辑，不提供官方截图。画风参考必须来自用户有权使用的素材，或由 GPT-image-2 生成并经用户认可的 `style-master.png`。

## 优先级

1. 角色所属具体动画、电影或系列版本。
2. 家族统一动画系列。
3. 默认简笔画 `visual-dna.md`。

禁止把同一家族的不同年代、2D、3D、定格、真人或游戏渲染混为一种“品牌画风”。

## 家族路由

| 家族 | 默认画风来源 | 路由规则 |
|---|---|---|
| POP MART | 默认简笔画 | 只有用户指定并提供某个官方动画或影像版本参考时才切换。手办、毛绒、动画必须分版本。 |
| Sanrio | Hello Kitty and Friends Supercute Adventures | 采用清晰平面轮廓、明亮纯色、简化室内外场景；若角色来自其他明确动画，改用该作品。 |
| Chiikawa | Chiikawa TV anime | 采用柔软细轮廓、浅色块、低对比背景、轻微日常喜剧节奏。 |
| Peanuts | Peanuts classic animated specials | 采用简洁黑色轮廓、平涂色块、克制背景和手绘动画节奏。用户指定新系列时可改用 The Snoopy Show 等版本。 |
| Moomin | 用户指定动画版本 | 先问经典 2D 或 Moominvalley 3D；未指定时采用经典 2D 书绘动画感，不混合 3D 材质。 |
| Jellycat | 默认简笔画 | 没有稳定统一动画版本时，保留毛绒材质身份，但使用默认简笔画构图与留白。 |
| MBTI / 16Personalities | 默认简笔画 | 没有统一动画；保留四大阵营配色和角色职业道具，使用默认简笔画。精确复刻官网人物插画需授权参考。 |

## 解释图适配

保留原动画视觉语言，但仍满足正文解释需求：一张一个机制、主体清晰、少量短标注、避免海报式标题。动画画风若背景很复杂，优先保留角色造型、轮廓与色彩，再简化环境，不牺牲信息可读性。
