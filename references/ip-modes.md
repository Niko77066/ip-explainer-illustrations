# IP 选择与角色锁定

## 选择模式

- 目录角色：从 `ip-catalog.md` 选择家族与官方英文角色名。
- 目录外角色：接受用户指定的同家族角色，先确认官方英文名与具体版本。
- 上传形象 Q 版：人物、宠物、玩偶或用户有权使用的角色。

不要凭角色名称编造具体系列服装。需要精确还原时，必须使用同版本参考图。

## 角色卡模板

```text
Official English name:
中文显示名：
家族：
具体系列或版本：
身份参考图：
轮廓与头身比：
脸部几何与五官数量：
主色及色块位置：
服装或配饰：
材质：
表情气质：
必须保留：
禁止新增或漂移：
```

角色卡必须包含可在图片上检查的事实。气质词只作补充，不能替代轮廓、比例、色块和部件描述。

## 上传图片转 Q 版

- 人物：保留脸型、发型、眉眼、眼镜、痣、主要服装和常用配饰；身体可缩成 2.5-3.5 头身。
- 宠物：保留耳形、毛色分区、眼睛、口鼻、尾巴和项圈；不要让品种刻板印象盖过个体特征。
- 玩偶或角色：保留轮廓、材质、眼口比例和标志性部件；简化细节但不改身份。
- 单张参考角度不足时，不虚构看不见的复杂标志；使用安全的正面或 3/4 视角。

先生成身份锚点，再做测试场景。锚点图本身不计入用户要求的正文配图数量，除非用户说要角色设定图。

## 一致性检查

- 缩略图轮廓是否一眼相同。
- 五官数量、耳朵、毛色与色块位置是否漂移。
- 主色、服装、配件和材质是否一致。
- Q 版比例是否稳定。
- 角色是否仍是动作主体。

若连续两张漂移，停止继续生成，先重做身份锚点或请求更清晰参考图。
